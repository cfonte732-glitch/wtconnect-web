# Variables — wtconnect-web

| Variable | Description |
|----------|-------------|
| `NEXT_PUBLIC_API_URL` | URL du Worker (ex. `https://api.votredomaine.com`) |
| `NEXT_PUBLIC_SUPABASE_URL` | Projet Supabase B — **auth uniquement** |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Clé anon — **auth uniquement** |

**Ne pas** définir `SUPABASE_SERVICE_ROLE_KEY` ni `DATABASE_URL` sur Pages.
